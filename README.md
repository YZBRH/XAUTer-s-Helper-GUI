## 前言

​	本项目是[YZBRH/XAUTer-s-UEAS_helper](https://github.com/YZBRH/XAUTer-s-UEAS_helper)的GUI版本，目前处于demo状态，正在极其缓慢开发中。



​	目前已实现功能：

- 自动登录及验证码识别

- 成绩查询

- 绩点计算

- 课程表查询

  

​	即将实现功能：

- 教程信息查询
- 自动选课
- 通过VPN登录教务系统
